# GQM

## 1. Objetivo (Goal)

* **Descrição do Objetivo:**

  * Exemplo: "Melhorar a qualidade do software entregue ao cliente em termos de confiabilidade e desempenho."

---

## 2. Perguntas (Questions)

Liste as questões que, ao serem respondidas, indicarão se o objetivo foi alcançado.

| ID  | Pergunta                                                                  |
| --- | ------------------------------------------------------------------------- |
| Q1  | Exemplo: "O sistema apresenta falhas críticas em produção?"               |
| Q2  | Exemplo: "Qual é o tempo médio de resposta das funcionalidades críticas?" |
| ... | ...                                                                       |

---

## 3. Métricas (Metrics)

Defina métricas mensuráveis para cada pergunta.

| Pergunta | Métrica                       | Fórmula / Unidades               | Fonte de Coleta             |
| -------- | ----------------------------- | -------------------------------- | --------------------------- |
| Q1       | Número de incidentes críticos | Contagem de incidentes/mês       | Ferramenta de monitoramento |
| Q2       | Tempo médio de resposta (ms)  | Soma dos tempos / número de reqs | Logs de aplicação           |
| ...      | ...                           | ...                              | ...                         |
